## 一、四部核心法律状态

### 《个人信息保护法》（PIPL，2021-11-01施行）

当前有效。核心条款已稳定。**配套规定仍在持续完善**：

- **个人信息保护合规审计管理办法**：2025-05-01 施行。企业须定期开展合规审计；满足条件的须委托专业机构审计。`[模型知识—待核验最新细则]`
- **个人信息保护负责人制度**：PIPL 第52条规定处理个人信息达到"规定数量"须指定负责人；网络数据安全管理条例第28条明确处理≥1000万人须适用重要数据规则。`[待核验最新门槛数字]`
- **自动化决策专项规定**：算法推荐管理规定（2022）和生成式AI服务管理暂行办法（2023）已施行，PIPL 自动化决策条款的细化规则仍在演进。`[待核验]`

### 《数据安全法》（DSL，2021-09-01施行）

当前有效。**重要数据目录体系仍在建立**：

- 国家层面重要数据目录尚未统一公布；行业主管部门目录已部分出台（工业和信息化、金融、汽车、测绘等）。`[待核验本行业目录状态]`
- 数据分类分级标准（GB/T 43697-2024）已发布，与 DSL 分类分级制度配套。`[模型知识—待核验是否已强制实施]`

### 《网络安全法》（CSL，2017-06-01施行，2022年修订）

当前有效。**等级保护制度（等保2.0）持续落地**：

- 《网络安全等级保护条例》（国家标准 GB/T 22239-2019）为现行标准。
- 《关键信息基础设施安全保护条例》（2021-09-01）明确 CIIO 范围和义务。`[待核验本行业 CIIO 认定最新标准]`

### 《网络数据安全管理条例》（2025-01-01施行）

**最新核心实施条例**，于 2024-09-24 颁布，2025-01-01 施行。是 PIPL/DSL/CSL 的统一实施细则，进一步明确：
- 大型网络平台（注册用户≥5000万或月活≥1000万）额外义务
- 生成式 AI 训练数据安全管理（第19条）
- 个人信息可携权的具体条件（第25条）
- 数据出境机制整合（第35-38条）

## 二、数据出境规则（变动频繁）

**最容易过时的领域，每次分析前须验证。**

### 安全评估申报门槛

依据《数据出境安全评估办法》（2022-09-01）及 2023年修订实施指南：

| 门槛 | 须申报安全评估 |
|---|---|
| CIIO 向境外提供任何数据 | ✓ |
| 处理≥100万人PI的处理者向境外提供 | ✓ |
| 累计向境外提供≥10万人PI（自然年起算） | ✓ |
| 累计向境外提供≥1万人敏感PI（自然年起算） | ✓ |
| 向境外提供重要数据 | ✓ |

- **有效期**：安全评估通过后有效期2年，届满需重新申报。`[待核验]`
- **免评估情形**：合同履行/劳动人力资源管理/法定义务/紧急保护。`[已在网络数据安全管理条例第35条明确，待核验最新解释]`

### 标准合同

依据《个人信息出境标准合同办法》（2023-06-01）：
- 适用：不触发安全评估门槛的个人信息出境
- 备案：须向省级网信部门备案（30天内）
- 模板：国家网信办已发布标准合同文本
- **个人信息出境标准合同备案指南第二版（2024-03-22）**：请验证最新备案要求是否有变化。`[待核验]`

## 三、个人权利请求响应时限

PIPL 无明确日历天数限制，规定"及时提供"（第45条）和"拒绝须说明理由"（第50条）。

- **网络数据安全管理条例第23条**：规定"及时受理，提供便捷支持"。
- 实践中通常参考15-30个工作日作为合理时限，但无统一法定期限。`[模型知识—待核验监管机构最新执法口径]`
- 个人权利请求的例外情形：PIPL 第18条规定的豁免。`[待核验]`
