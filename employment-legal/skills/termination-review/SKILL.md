# /termination-review

1. 加载 `~/.claude/plugins/config/claude-for-legal/employment-legal/CLAUDE.md` → 解除审查触发器、高风险标志、经济补偿金做法、省市规则。
2. 使用以下工作流。
3. 遍历检查清单。检查每个高风险标志。
4. 按解除方式审查：协商一致/用人单位单方/试用期/经济性裁员。
5. 如果任何高风险标志触发：在继续之前按表升级，不要在没有签字的情况下继续。

## 目的

大多数解除都没问题。少数是等待发生的诉讼。此 skill 运行在决策最终之前捕获第二种类型的检查清单。Skill 不陈述法律——每个省市特定规则在审查时进行研究并引用。

**解除合法性三要素（举证责任在用人单位）：**
1. **事实依据** — 解除理由是否客观存在？
2. **制度依据** — 规章制度是否经民主程序制定且已公示告知员工？
3. **程序依据** — 是否依法通知工会？是否有书面解除通知？是否留有送达证据？

## 工作流

### 步骤 1: 基本事实

- 员工姓名（或如果保持抽象则为岗位）
- 工作地点（省市——这决定适用哪些地方性规则和仲裁管辖）
- 解除原因（绩效、严重违纪、客观情况重大变化、经济性裁员、协商一致）
- 入职时间和工龄（计算经济补偿金N）
- 是否为"三期"女职工（孕期/产期/哺乳期）
- 是否处于医疗期、工伤认定期、工伤医疗期
- 是否为工会成员或工会代表（特殊保护）
- 计划的解除日期是什么时候

### 步骤 2: 高风险标志扫描

这是最重要的步骤。检查 `~/.claude/plugins/config/claude-for-legal/employment-legal/CLAUDE.md` 中的每个标志。默认集合：

| 标志 | 为什么高风险 | 检查 |
|---|---|---|
| **三期女职工** | 《劳动合同法》第42条绝对禁止解除 | 员工目前是否处于孕期/产期/哺乳期？ |
| **医疗期员工** | 《劳动合同法》第42条禁止解除 | 员工目前是否在依法享受医疗期？ |
| **工伤认定/医疗期** | 《劳动合同法》第42条禁止解除 | 员工是否处于工伤认定期间或工伤医疗期未满？ |
| **最近提出仲裁/举报/信访** | 报复认定风险 | 员工最近是否提起劳动仲裁、向劳动监察部门举报或信访？ |
| **举报违法行为** | 打击报复风险，保护举报人 | 员工是否向监管机构或公司合规渠道举报过违法行为？ |
| **违纪证据不足** | "为什么现在"问题 | 对于违纪解除：规章制度是否经民主程序+公示？违纪事实是否有书面记录？处分程序是否完整？ |
| **规章制度程序瑕疵** | 制度无法约束员工 | 规章制度是否经职工代表大会/全体职工讨论+与工会协商+公示告知？ |
| **未通知工会** | 程序违法 | 是否依法提前书面通知工会并听取意见？（解除前须通知） |
| **竞业限制员工** | 须继续支付补偿金 | 员工是否签有竞业限制协议？解除后是否仍须支付竞业限制补偿金？ |
| **比较问题** | 差别对待风险 | 其他人是否在做同样的事情而不被解除？ |

**法定禁止解除情形（《劳动合同法》第42条）：**
以下情形用人单位不得依据第39条（严重违纪等）、第40条（客观原因）解除劳动合同：
1. 从事接触职业病危害作业的劳动者未进行离岗前职业健康检查，或者疑似职业病病人在诊断或医学观察期间的
2. 在本单位患职业病或者因工负伤并被确认丧失或者部分丧失劳动能力的
3. **患病或者非因工负伤，在规定的医疗期内的**
4. **女职工在孕期、产期、哺乳期的**（三期保护）
5. 在本单位连续工作满十五年，且距法定退休年龄不足五年的
6. 法律、行政法规规定的其他情形

**任何上述情形触发 → 在解除继续之前按 `~/.claude/plugins/config/claude-for-legal/employment-legal/CLAUDE.md` 升级。** 不是之后。之前。

**经济性裁员高风险标志。** 当以下所有为真时触发此标志：
1. 解除原因为"经济性裁员"——并且
2. 未按《劳动合同法》第41条程序进行：提前30日向工会或全体职工说明情况→向劳动行政部门报告→优先留用特定人员

> 🔴 **经济性裁员程序缺陷** — 经济性裁员须：①符合法定情形（企业破产重整/生产经营困难/客观情况重大变化等）；②提前30日向工会或全体职工说明情况并听取意见；③向劳动行政部门报告裁员方案；④优先留用无固定期限合同员工、家庭无其他就业成员且有需赡养老人或未成年子女员工。程序不合规的裁员可能被认定为违法解除，须支付2N赔偿金。

### 步骤 3: 解除方式与省市特定要求

> **在最终确定计划之前研究员工工作省市的适用规则。** 具体来说：
>
> - **解除方式选择** — 协商一致解除（第36条）、用人单位单方解除（第38、39、40、41条）、劳动合同到期终止（第44条）。不同方式的经济补偿金义务不同。
> - **工会通知** — 《劳动合同法》第43条要求用人单位单方解除前书面通知工会。研究当地具体要求和时限。
> - **送达要求** — 解除通知须有送达证据（直接送达+签收、邮寄送达+回执、公告送达等）。无送达证据是败诉的重要原因。
> - **经济性裁员通知** — 若裁员20人以上或占职工总数10%以上，须提前30日向工会或全体职工说明，向劳动行政部门报告。
>
> 引用主要来源（具体法律条文）。验证时效性（特别是各地高院指导意见）。
>
> **没有静默补充。** 如果对配置的法律研究工具（元典）的研究查询对于省市的特定规则返回很少或没有结果，报告发现的内容并停止。不要未经询问就从网络搜索或模型知识填充空白。
>
> **来源归因。** 用其来源标记计划中的每个引用——解除规则、工会通知要求、送达规则——来自：从元典法条检索工具获取的引用标记 `[元典法规]`；从元典案例检索工具获取的引用标记 `[元典案例]`；网络搜索引用为 `[网络检索 — 需核验]`；从训练数据回忆的引用为 `[模型知识 — 需核验]`；用户提供的引用为 `[用户提供]`。

### 步骤 4: 经济补偿金与解除协议

根据 `~/.claude/plugins/config/claude-for-legal/employment-legal/CLAUDE.md` → 标准经济补偿金：

**经济补偿金计算（《劳动合同法》第46-47条）：**
- 基数：劳动者在本单位工作的年限×月工资（离职前12个月平均工资）
- 满1年支付1个月；满6个月不满1年按1年计；不满6个月支付0.5个月
- 月工资高于当地上年度职工月平均工资3倍的，按3倍封顶，且最多支付12年

**违法解除赔偿金（《劳动合同法》第48条）：**
- 赔偿金 = 经济补偿金 × 2（即2N）
- 与经济补偿金二者选其一，不能叠加

**解除协议内容检查（如适用）：**
- 是否包含解除原因、解除日期、经济补偿金金额及支付安排
- 是否包含竞业限制条款（若有）
- 是否明确社保、公积金截止及账户处理
- 是否包含工作交接安排
- 劳动者是否真实自愿（意思表示是否真实——胁迫/欺诈可撤销）

> **研究适用的省市解除协议要求。** 部分省市对解除协议的形式、内容有特殊要求；部分高院认为解除协议须满足特定条件才能对抗劳动者的撤销请求。不要凭记忆陈述——研究具体情况下的当前有效规则并引用主要来源。

### 步骤 5: 文档检查

特别是对于绩效/违纪解除：

- 是否有书面记录？书面警告、绩效改进计划（PIP）、考核不达标记录？
- 规章制度的解除依据是否明确（列明何种行为构成"严重违纪"）？
- 规章制度的民主程序和公示告知有无证据？
- 书面记录是否讲述了一致的故事？
- 最近是否有正面评价、晋升、奖金等与解除理由矛盾的记录？

"为什么现在"问题：如果此人绩效不达标已经一年，什么改变了？答案应该被记录。

## 后果行动关卡（解除劳动合同）

**在生成"可以继续"建议或标记为准备就绪的解除日检查清单之前：** 阅读 `~/.claude/plugins/config/claude-for-legal/employment-legal/CLAUDE.md` 中的 `## Who's using this`。如果角色是 **非律师**：

> 解除劳动合同有法律后果——违法解除须支付赔偿金（2N），可能引发劳动仲裁及诉讼。你是否已与律师审查此解除？如果是，继续。如果不是，这是带给他们的简报：
>
> - 员工、工作地点（省市）、解除方式、计划日期
> - 解除合法性三要素检查结果（事实依据+制度依据+程序依据）
> - 每个高风险标志的详情（三期/医疗期/工伤/最近仲裁/举报等）
> - 省市特定发现（工会通知、送达要求、经济性裁员报告）及其引用来源
> - 经济补偿金/赔偿金计算（含封顶规则）
> - 未决问题和未解决内容
> - 可能出错的地方（此事实模式支持的仲裁请求理论）
> - 要问律师什么（这是否是合法解除；我们是否需要首先补充文档；工会通知程序是否完整）
>
> 如果你需要找到律师：联系当地律师协会或省级律师协会——大多数提供律师推荐服务。劳动争议领域的律师在解除前的一两小时咨询费用，通常远低于仲裁和诉讼的赔偿金。

## 此 skill 不做什么

- 做出解除决策。它检查决策。
- 进行谈判和对话。经理和 HR 做那个。
- 从记忆陈述工会通知时限或省市规则——每个规则在审查时进行研究并引用。
- 保证没有诉讼。它通过捕获明显问题来降低风险。
