# /policy-drafting

1. 加载 `~/.claude/plugins/config/claude-for-legal/employment-legal/CLAUDE.md` → 省市足迹、员工手册位置。
2. 使用以下工作流。
3. 起草核心规章制度。检查足迹中每个省市是否需要变体。
4. 输出：核心制度 + 省市补充。标记法律目前正在变化的领域。

## 目的

适合上海的规章制度可能在广东有不同的合规要求。此 skill 起草核心规章制度并在足迹要求不同规则时生成省市补充。

**重要前提——民主程序与公示告知（《劳动合同法》第4条）：**
用人单位在制定、修改或者决定有关劳动报酬、工作时间、休息休假、劳动安全卫生、保险福利、职工培训、劳动纪律以及劳动定额管理等直接涉及劳动者切身利益的规章制度或者重大事项时，应当经职工代表大会或者全体职工讨论，提出方案和意见，与工会或者职工代表平等协商确定。规章制度应当公示，或者告知劳动者。**未经民主程序制定和公示告知的规章制度，不能作为处理劳动争议的依据。**

`~/.claude/plugins/config/claude-for-legal/employment-legal/CLAUDE.md` → 省市足迹、员工手册位置和格式。

## 工作流

### 步骤 1：确定规章制度范围

- 规章制度目的是什么？（加班管理、年休假、远程办公、社交媒体、纪律处分等）
- 为什么现在需要？（法律要求、事件、增长、发现差距）
- 适用于谁？（所有员工、特定岗位、特定地点）
- 是否需要经过民主程序？（直接涉及劳动者切身利益的必须）

### 步骤 2：省市扫描

对于足迹中的每个省市，检查：此省市是否对此主题有特定规则？

**有省市差异的常见主题：**

| 主题 | 差异 |
|---|---|
| 带薪年休假 | 国家规定为基础，各省市可有更高标准；实施细则有差异 |
| 加班管理 | 加班费基数认定、加班申请程序各省市高院裁判口径不同 |
| 工时制度 | 综合计算工时制/不定时工时制须经劳动行政部门审批——各省市程序不同 |
| 女职工保护 | 三期女职工保护（产假天数+省市生育奖励假各地不同） |
| 医疗期 | 国家规定为基础，各地可有更高要求 |
| 竞业限制 | 补偿金比例各省市下限不同 |
| 经济补偿金封顶 | 各省市封顶基数（当地上年度职工月平均工资）不同 |
| 最低工资标准 | 各省市不同，通常每年更新 |

如果主题没有省市差异（比如着装规范），跳过此步骤。

### 步骤 3：起草核心规章制度

一个制度。适用于所有地方。清晰可读——员工应该不需要律师就能理解。

**结构：**
- 目的（一句话——此制度为何存在）
- 适用范围（适用于谁）
- 规则（什么是要求的/允许的/禁止的）
- 程序（如何申请、谁批准、违反后果）
- 附则（与其他制度的关系；联系方式）

**注意事项：**
- 违纪处分条款须具体明确（"严重违纪"须列明具体情形），避免用"情节严重"等模糊表述
- 违纪处分的后果须明确列明（书面警告、降职、解除等），且解除须符合"严重违纪"认定
- 不得设置违法内容（如超过法定加班时限要求、变相降低法定福利）
- 避免使用："前述"、"不限于此"等不具体表述；"视情况而定"等不可操作表述

避免使用法律术语堆砌。这是规章制度，不是合同。

### 步骤 4：省市补充

对于规则不同的每个省市，一个补充：

```markdown
### [省市] 补充

在[省市]工作的员工除核心制度外，还需遵守以下内容（或替代）：

- [具体差异]
- [如有帮助，引用省市法律或高院指导意见]
```

保持补充简洁。只写不同的部分——不要重复核心内容。

### 步骤 5：交叉检查

- 此制度是否与员工手册中已有的内容冲突？
- 它是否承诺了超过公司打算交付的内容？（规章制度是承诺——法院按员工手册承诺约束用人单位。）
- 违纪处分内容是否符合比例原则（处罚与违纪行为须相称）？
- 民主程序建议：在正式发布前，应当经过民主程序（职代会或全体员工讨论），并公示或告知员工。

## 输出

```markdown
# [规章制度名称]

## 核心制度

[全文]

## 省市补充

### [省市 1]
[补充]

### [省市 2]
[补充]

---

## 起草说明（内部——插入员工手册前删除）

- **省市扫描：** [检查了哪些省市，哪些有差异]
- **民主程序建议：** [本制度是否涉及劳动者切身利益，建议走民主程序后再发布]
- **与现有员工手册冲突：** [无 | 列表]
- **法律正在变化：** [任何省市处于变化中或有最新政策]
- **审查节奏：** [何时重新审视——年度，或当 X 发生时]
```

> **草稿，不是生效的规章制度。** 这是供律师审查的起草辅助工具，不是你可以直接发布的制度。发布规章制度有法律后果——涉及劳动者切身利益的规章制度须经民主程序制定，且须公示告知员工，否则不能作为处理劳动争议的依据。由具备相应资质的劳动法律师审查、按需编辑，经民主程序后再在公司推行。不要在未审查的情况下直接发布或分发此草稿。

## 交接

到 handbook-updates skill：当此制度获批准时，它将与当前员工手册进行差异比较并标记变更。

## 此 skill 不做什么

- 批准规章制度。它起草；人批准，民主程序决定。
- 推行制度。向员工传达是 HR 工作流（且须满足公示告知要求）。
- 覆盖中国所有省市——仅覆盖足迹中的。如果足迹扩展，重新运行。
